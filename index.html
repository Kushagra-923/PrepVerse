Q1: The Startup Logistics Problem

Lina is the operations manager at a fast-growing startup. To streamline operations, she must solve a few problems using functions.

Challenge 1: Delivery Routes

  • If Lina assigns a driver to a unique route, but some customers are missed, what kind of function is this (Injective, Surjective, or Bijective)?
  • If every customer is covered but some drivers have more than one route, what kind of function is this (Injective, Surjective, or Bijective)?

Challenge 2: Pricing Model

  • Service A: f(x) = 5x + 15 and Service B: g(x) = 3x
  • If a client subscribes to both services, what is (f+g)(x)?
  • If Lina applies a discount by multiplying costs, what is (f∗g)(x)?

Challenge 3: Efficiency Analysis

  • Delivery time: g(x) = x + 2, Cost: f(x) = x + 5
  • Find (f∘g)(x) and (g∘f)(x).

Challenge 4: Reversing a Process

  • Given h(x) = (2x + 4) / 3, find h⁻¹(x).
Answer: 1. Not surjective, but injective. 2. Surjective, not injective. 3. (f+g)(x) = 8x + 15, (f∗g)(x) = 15x + 5x² 4. (f∘g)(x) = (x+2)+5 = x+7, (g∘f)(x) = (x+5)+2 = x+7 5. h⁻¹(x) = (3x - 4) / 2

Q2: Rules of Inference Problem

Hypotheses: 1. If it does not rain or if it is not foggy, then the sailing race will be held and the lifesaving demonstration will go on. 2. If the sailing race is held, then the trophy will be awarded. 3. The trophy was not awarded.

Prove: "It rained".

Answer: Using rules of inference: - From (2) and (3), sailing race was not held. - From (1), for the race not to be held, "it must have rained". Therefore, conclusion: It rained.

Q3: Set Operations

A = {2, 4, 6, 8}, B = {4, 6, 9, 12}, C = {6, 12, 15}

  • (A∪B)∩C
  • (A−B)∪(B∩C)
  • A−B−C
  • A−(B∩C)
Answer: a. (A∪B)∩C = {6,12} b. (A−B)∪(B∩C) = {2,8,6,12} c. A−B−C = {2,8} d. A−(B∩C) = {2,4,8}

Q4: The Digital Vault Challenge

A hacker named Alex designed a vault that requires solving binary and logical challenges.

Challenge 1: Code A = 10101000, Code B = 11000101. Find A+B in decimal and 8-bit binary.

Challenge 2: Find the 16-bit two's complement of (-8).

Challenge 3: P = ((x ∧ y) ∧ (x ∨ y)) ∧ (x ∧ y), where x=15 and y=9. Find P in decimal.

Answer: 1. A=168, B=197 → A+B=365, Binary=101101101 (needs 9 bits). 2. 16-bit two's complement of -8 = 1111111111111000. 3. x=15 (1111), y=9 (1001) → x∧y=1001=9. x∨y=1111=15. So P = (9 ∧ 15) ∧ 9 = 9. Decimal = 9.

Q1: Compute A^B

You are given two integers A and B. Find the value of A^B.

Answer: A^B can be computed using Math.pow(A, B). Example: For A=2, B=3 → 2^3 = 8

Q3: Relation of N with 18

Given a number N, print whether N is greater than, smaller than, or equal to 18.

Answer: if (N > 18) print "N is Greater than 18" else if (N < 18) print "N is smaller than 18" else print "N is equal to 18"

Q4: Leap Year III

Given an integer A (year), return 1 if it is a leap year, else return 0.

Answer: Leap year if: - divisible by 400 OR - divisible by 4 AND not divisible by 100 Example: 2020 → 1, 1999 → 0

Q5: Is It Perfect?

Check if a number N is a perfect number.

Answer: Perfect number = sum of its proper divisors. Example: 6 → divisors (1,2,3), sum=6 → YES 4 → divisors (1,2), sum=3 → NO

Answer: Example: N=3 1 2 3 1 2 1

Q1: Build a simple webpage with a navigation bar

Description: Create a simple webpage with a navigation bar using the <nav> tag that contains three links inside anchor tags: "Home", "About", and "Contact". The links should be aligned horizontally and when hovered over they should change color to green.

Test cases: Should have links present and color changes upon hover.

Answer: Use <nav> with <a> tags styled using CSS flex or inline-block. Apply hover state to change color.

Q2: Style a Button with Rounded Corners

Description: Create a button with the label "Submit" that has rounded corners, a blue background, and white text. When hovered, the background should change to dark blue.

Test cases: Should have button with rounded corners and color changes on hover.

Answer: Use CSS properties like border-radius, background-color, and hover selector for styling.

Q3: Create a Responsive Two-Column Layout

Description: Create a responsive webpage with two columns: the left column should have a width of 30% and the right column 70%. On screens smaller than 600px, the two columns should stack vertically.

Test cases: Should display two columns on large screens and stack them on a small screen.

Answer: Use CSS Flexbox or Grid with media queries for responsiveness.

Q4: Implement a Basic Box Model Structure

Description: Create a box with a width of 300px, height of 150px, a blue border of 5px, 20px of padding, and 10px of margin. The background of the box should be yellow.

Test cases: Should have a box with the correct dimensions, padding, margin, and border.

Answer: Apply width, height, margin, padding, border, and background-color styles in CSS.

Q5: Create a Flexbox-Based Pricing Table

Description: Design a pricing table with three columns for different plans: Basic, Standard, and Premium. Each column should have a header, a list of features, and a "Buy Now" button. The columns should be centered on the page and have equal width.

Test cases: Should have a container with centered columns and equal width.

Answer: Use Flexbox with justify-content: center; and flex: 1; for equal column widths.
